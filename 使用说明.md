#### 使用说明

1. 运行链端
```
cd nodes/127.0.0.1/

Ubuntu：
chmod +x start_all.sh
Centos7：
chmod 777 start_all.sh
chmod 777 fisco-bcos

bash start_all.sh
```

2. 启动服务端
```
cd server/
npm install
npm start
```

3. 启动客户端
```
cd client/
npm install
npm start
```
